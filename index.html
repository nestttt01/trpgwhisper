新手教學與遊戲指南

第一次玩，從這裡開始

你只需要完成 API 驗證、建立角色，然後輸入第一個行動就能開始。角色面板、冒險日誌與其他進階功能，可以等實際需要時再使用。

1. 這是什麼？

這是一個單人 AI TRPG／劇情模擬器。你可以建立自己的角色、NPC 與世界觀，讓 AI 擔任 DM、旁白及其他角色，陪你進行冒險、戀愛、日常、懸疑或任何想玩的故事。

2. 驗證 API Key

開始前，請先準備 Google AI Studio 或 OpenRouter 的 API Key。貼上金鑰並完成驗證後，選擇一個 AI 模型即可進入遊戲。想節省用量可以先選擇輕量模型；重視劇情與角色演出，可以優先嘗試清單中標有 ★ 的模型。API Key 不會包含在匯出備份中。

3. 建立角色與故事

在「劇本創角」中設定玩家名稱、頭像、能力值與人物背景，也可以建立多位 NPC。接著設定故事的世界觀與情境；也能保留人物只隨機產生世界，或讓 AI 將人物與世界全部隨機生成。生成結果會先預覽，套用後仍需按下儲存，不會直接覆蓋原設定。

4. 怎麼玩？

進入遊戲後，直接在下方輸入角色想說的話或想做的事。你也可以點選 AI 提供的選項。若行動具有風險，可以使用骰子按鈕進行 D20 判定；系統會依角色能力、難度與骰點決定成功或失敗。NPC 只有在劇情明確確認時才會死亡；標準模式可由劇情或「神」復活，困難模式每次死亡只有一次復活檢定，極限模式則永久死亡。

5. 玩家、旁白與「神」模式

一般輸入會被視為玩家角色的行動。輸入 ［切換旁白］ 可以改用場外視角安排環境、鏡頭與 NPC 行動；輸入 ［切換玩家］ 就能回到玩家角色。「神」按鈕則是創作者模式，可以直接調整場景、時間、人物或事件。神模式的內容不會被 NPC 當成玩家說出口的話。

6. 角色面板與冒險日誌

角色面板可以查看與修改玩家狀態、NPC、道具、Flags、任務、劇情摘要及角色關係。AI 每回合會優先參考這些重點資料，幫助故事保持連貫。完整的歷史事件則保存在「冒險日誌」，可以搜尋、分頁、編輯或整理，不會因為摘要更新而自動消失。

7. 存檔與備份

每個遊戲紀錄會分開保存，不會和其他故事混在一起。資料主要保存在目前裝置的瀏覽器，不會自動同步到其他手機或電腦。建議定期在存檔選單按「匯出」保存備份；更換裝置時，再使用「匯入」恢復資料。

💡 AI 忘記事情怎麼辦？

打開角色面板，檢查重點劇情、任務、角色關係與 Flags 是否正確。你可以手動修正，或使用「整理摘要」重新整理目前狀態。

💡 劇情重複或場景錯亂怎麼辦？

先確認摘要與冒險日誌是否正確。如果 AI 仍然混淆，可以開啟「神」模式，直接說明目前場景、在場角色與接下來希望聚焦的方向。

💡 AI 一直沒有回應怎麼辦？

系統遇到暫時性連線問題時會自動重試一次。若等待太久，可以按「取消等待」停止本次請求，再檢查網路、API 額度或更換模型。尚未完成的輸入會盡量放回輸入框。
